### Annotations Layer

Create a separate Figma layer group called `📋 Annotations`. Inside it, place red sticky note components (use Figma's built-in sticky or a red-filled frame with white text) at these points:

- On the Log form's location field: *"No text input for location. Geolocation API only — prevents manual spoofing."*
- On the Submit button disabled state: *"Button stays disabled until navigator.geolocation.getCurrentPosition() resolves with coordinates."*
- On the Latitude/Longitude columns in the table: *"Coordinates stored in DB from API — not editable by user."*
- On the E-Waste metric card: *"Live aggregate: SUM(weight) WHERE category = 'Electronic' — updates on each new log."*
- On the dashboard table: *"Fetches all rows from waste_logs table in Supabase/PostgreSQL — paginated at 50 rows."*
